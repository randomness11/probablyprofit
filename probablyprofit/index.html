Polymarket & Kalshi bots in plain English.

probablyprofit is a framework for traders who have the strategy but don't want to spend 6 hours debugging an API wrapper. Define your edge in a sentence; we handle the execution.

Get Started
$ pip install probablyprofit click to copy

# poll-driven election arb

use polls from "538", "rcp", "nyt"
use sentiment from "twitter", "polymarket"
use odds from "betfair"

watch "Trump Winner", "DeSantis Winner"
watch "GOP Nominee", "Senate Control"

set portfolio_max = $10k
set conviction = avg(polls) - price

# fade the public when data disagrees
when conviction > 0.15 and sentiment < -0.2
  if odds > price + 0.10
    buy conviction * $1000 of YES
    hedge with "Senate Control" at 20%

# correlated momentum
when "GOP Nominee" moves 5% in 1h
  mirror move on "Trump Winner"
  notify "correlation trigger"
